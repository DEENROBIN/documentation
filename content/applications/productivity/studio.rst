======
Studio
======

Studio is a toolbox that allows you to customize Odoo without coding knowledge. For example, you
can, on any app, add or modify:

- :ref:`Fields <studio/fields>`
- :ref:`Views <studio/views>`
- :ref:`Models <studio/models-modules-apps>`
- :ref:`Automated actions <studio/automated-actions>`
- PDF reports
- Approval rules
- Security rules

Or you can :ref:`build an app from scratch <studio/models-modules-apps>`.

.. seealso::
   - `Odoo Tutorials: Studio <https://www.odoo.com/slides/studio-31>`_
